Tipos de Selectores CSS

Nombre del selector Que selecciona Ejemplo
Selector de elemento Todos los elementos HTML h1 selecciona los titulos h1
Selector de identificacion ID elemento en la pagina con ID especificado #mi-id selecciona h1 id="mi-id" y p id="mi-id"
Selector de clase elementos en la pagina con la clase determinada .mi-clase selecciona h1 class="mi-clase" y p class="mi-clase"
Selector de atributo los elementos en la pagina con atrbuto especificado img[src] selecciona img src="mi-imagen" pero no img
Selector de pseudoclase los elementos especificados, pero solo cuando este en el estado especifico, ejemplo cuando el puntero esta sobre el a: hover selecciona a, pero cuando el puntero esta sobre el enlace
